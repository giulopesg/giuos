# /sprint-close

Encerra o ciclo Shape Up e prepara o cooldown.

**O agente:**
1. Resume o que foi entregue neste ciclo
2. Lista o que ficou fora (no-gos respeitados)
3. Aponta dívida técnica intencional (se houver)
4. Propõe retro rápida: o que aprendemos? o que mudou na visão do produto?
5. Atualiza SPEC.md com `status: closed` e PITCH.md com `status: closed`
6. Sugere timing para próximo shaping
