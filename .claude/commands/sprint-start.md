# /sprint-start

Inicia o Building de um ciclo Shape Up.

**Pré-condição:** `docs/SPEC.md` com `approved: true` (assinada por Giuliana)

**O agente:**
1. Confirma que a SPEC está aprovada
2. Lê o stack pack ativo
3. Declara o que vai construir e em qual ordem
4. Aguarda confirmação de Giuliana
5. Ativa o `build-agent`
6. Primeiro passo: verifica referências Figma para componentes UI
