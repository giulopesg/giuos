# /new-project

Inicia um novo projeto no GiuOS.

**O agente:**
1. Pergunta o modo de entrada:
   > "Estamos no início de uma ideia que ainda precisa de discovery, ou você já sabe o que quer construir?"
2. Modo Exploratório → Estágio 1 do WIZARD (Discovery com Design Thinking)
3. Modo Execução → Estágio 3 do WIZARD (PRD direto)
4. Preenche o bloco de projeto no `CLAUDE.md`
5. Cria `session-log/INDEX.md` se não existir
6. Cria `docs/` se não existir
